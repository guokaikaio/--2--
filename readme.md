### 官方文档

过孔主要分为两种：
1、沉铜孔PTH（Plating Through Hole），孔壁有铜，一般是过电孔（VIA PAD）及元件孔（DIP PAD）。
2、非沉铜孔NPTH(Non Plating Through Hole)，孔壁无铜，一般是定位孔及螺丝孔。
